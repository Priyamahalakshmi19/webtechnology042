Lab Manual

Below is the list of lab experiments for Web Technology 42. Each experiment helps you apply concepts in HTML, CSS, JavaScript, and more!

  1. Experiment 1: HTML Page creation
    Aim: Create a simple HTML page with headings, paragraphs, and lists.
    <!DOCTYPE html>
<html>
<head>
  <title>My First HTML Page</title>
</head>
<body>
  <h1>Welcome</h1>
  <p>This is a basic HTML page.</p>
</body>
</html>
  2. Experiment 2: Formatting in html
    Aim: Use text formatting tags and create internal and external links.
    <p><b>Bold Text</b> and <i>Italic Text</i></p>
<a href="https://example.com" target="_blank">Visit Example</a>
  3. Experiment 3: Tables and lists
    Aim: Create a page using tables and embed images.
    <table>
  <tr>
    <th>Header 0</th>
    <th>Header 1</th>
  </tr>
  <tr>
    <td>Data 3</td>
    <td>Data 4</td>
  </tr>
</table>
